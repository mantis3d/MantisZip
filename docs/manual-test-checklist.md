# MantisZip 手动测试清单

> 测试人员逐条执行操作，在"实际结果"列填写观察到的行为，在"状态"列标记 ✅（通过）或 ❌（失败）。全部完成后汇总缺陷。

**备注列标注说明**:
- `[7z]` — 需已安装 7-Zip（`C:\Program Files\7-Zip\7z.exe`）
- `[no7z]` — 未安装 7-Zip 的环境
- `[cascade]` — 右键菜单为"层叠模式"（设置 → 右键菜单 → 启用层叠菜单）
- `[verb]` — 右键菜单为"动词模式"（默认，层叠菜单关闭）
- `[win10]` / `[win11]` — 特定操作系统行为
- `[default]` — 全部默认设置
