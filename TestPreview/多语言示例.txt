--- English ---
MantisZip is a free and open-source file compression/decompression software for Windows.
It supports popular formats including ZIP, 7z, TAR, GZ, and RAR.
Built with .NET 9 + WPF, assisted by Sisyphus Agent.

====================================
  功能特性 | Features
====================================

✓ 文件内预览（图片 / 文本 / HTML / Markdown）
✓ 密码管理器（保存常用密码，glob/regex 匹配）
✓ 快速密码验证（读 1 字节即可确认密码）
✓ 解压冲突处理（覆盖/重命名/跳过/询问）
✓ Shell 右键菜单 + 文件关联
✓ 拖拽提取到 Explorer
✓ CLI 命令行操作
✓ 分卷压缩
✓ AES-256 加密

  当前版本: v0.2.2
